# Furigana
网易云音乐插件：给日文歌词的汉字标注假名<br>
使用了kuroshiro的在线API：<br>
https://kuroshiro.org/

<font size=6 color=#FF0000>由于Kuroshiro的在线API似乎不能用了，所以最主要的功能失效了。争取2024年春节前更新一个新版本，把API换掉</font>

1. 修复了一些小bug
2. 部分兼容了网易云音乐2.x版本
3. 兼容了Apple Music-like lyrics
